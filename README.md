# P1ElektroniskValg
Studieprojekt

Dette produkt for projektet er opdelt i 2 bestanddele. 

Vote.c tillader brugere at stemme til et elektronisk folketingsvalg og strukturere denne data i en tekstfil. 

Encryption.c behandler den indsamlede data og krypterer denne ved hjælp af one time pad. Efter et bestemt tidspunkt, som kan predefineres af en administrator, er det muligt at lade programmet dekryptere den krypterede data.
